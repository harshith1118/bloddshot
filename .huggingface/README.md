# 🩸 GutCheck - Blood Test Analyzer

**Your blood test, finally explained.**

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://gutcheck.hf.space)

A AI-powered blood test analyzer built with Mistral AI for the **Mistral AI Worldwide Hackathon 2026**.

## ✨ Features

- 📄 **PDF Upload** - Upload your blood test report PDF
- 🔬 **AI Analysis** - Mistral Large 3 analyzes all biomarkers
- 🎯 **Clear Status** - 🟢 Normal / 🟡 Borderline / 🔴 Concern flags
- 💡 **Plain English** - No medical jargon, easy to understand
- 📋 **Action Plan** - Personalized diet & lifestyle recommendations
- 🔊 **Voice Output** - Results read aloud by Voxtral
- 🔍 **Deep Dive** - Research agent finds latest studies

## 🚀 Try It

1. Upload a blood test PDF (or use our sample reports)
2. Click "Analyze Report"
3. Get instant results with explanations and recommendations

## 🛠️ Built With

- **Frontend:** Streamlit
- **AI Model:** Mistral Large 3 (`mistral-large-latest`)
- **Voice:** Voxtral (`voxtral-v0-2507`)
- **PDF Parsing:** PyMuPDF + pdfplumber
- **Research:** Mistral Agents API + web_search
